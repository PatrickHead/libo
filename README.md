## Introduction

libo provides functions that can read, write, create and modify Office documents in memory and files.

NOTE:  Only Excel (XLSX) files are implemented

<a id="installation"></a>
## Installation

In the spirit of GNU Autotools, follow these steps to install libo:

  -# Obtain libo.txz
  -# tar xf libo.txz
  -# cd libo
  -# make
  -# sudo make install

<a id="library"></a>
## Library

A static library named libo.a should be installed in /usr/local/lib after running `sudo make install`
